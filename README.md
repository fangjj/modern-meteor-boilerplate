# Modern Meteor Boilerplate

## Try

### Install Meteor

```bash
curl https://install.meteor.com/ | sh
```

### Install npm packages

```bash
meteor npm i
```

### Start

```bash
npm run start
```

### Visit

Open `http://127.0.0.1:3000` at browser.
